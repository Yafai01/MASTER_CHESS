# 👑 Master Chess: Royal Edition

An elegant, story-driven chess learning platform designed to take you from a novice to a grandmaster.

![Master Chess Banner](https://images.unsplash.com/photo-1528819622765-d6bbf132f793?auto=format&fit=crop&q=80&w=1200)

## 🌟 Overview

Master Chess: Royal Edition is a premium web application that combines a professional chess engine with a curated learning path. Immerse yourself in the world of chess with our "Royal" aesthetic, featuring gold-toned UI, smooth animations, and a comprehensive curriculum.

## ✨ Features

- 📖 **Story Mode**: A 6-chapter journey covering the board, pieces, tactics, strategy, and endgames.
- 🧩 **Interactive Puzzles**: Over 20 curated puzzles (Mate in 1, Mate in 2, and Tactics) with real-time feedback.
- ♞ **Play Mode**: Challenge our custom AI engine with multiple difficulty levels and a professional move history tracker.
- 🏆 **Progress Tracking**: Track your XP, lesson completion, and climb the global leaderboard.
- 🔊 **Sound Engine**: Custom Web Audio procedural sounds for moves, captures, and checkmates.
- 🎨 **Premium UI**: Dark-mode-first design with Cinzel typography and elegant gold shimmer effects.

## 🛠️ Technology Stack

- **HTML5 & CSS3**: Custom styles with a focus on premium aesthetics and responsive design.
- **JavaScript (ES6+)**: Pure vanilla JS logic for the game engine and app state.
- **Chess.js**: Robust move validation and game state management.
- **Web Audio API**: Real-time sound generation.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Yafai01/MASTER_CHESS.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser. No installation required!

## 📜 Curriculum

1. **Chapter 1**: The Board & Setup — Understanding the battlefield.
2. **Chapter 2**: The Army — Learning how every piece moves and captures.
3. **Chapter 3**: First Battles — Basic rules and common pitfalls.
4. **Chapter 4**: Tactics — Mastering forks, pins, and skewers.
5. **Chapter 5**: Strategy — Understanding openings and middle-game planning.
6. **Chapter 6**: Endgames — How to bring the game to a royal finish.
